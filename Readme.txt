== Description ==

WP Adcenter Lite is a complete WordPress advertising plugin.You can have Unlimited number of Adzones and place banners anywhere on site.

**Requirements:** WordPress 4.0+, PHP5.3 and higher

**Important Links:**

* [More Details](http://wpadcenter.com/)

**Lite Version:**

This Lite version of WP Adcenter Plugin does not include all the features like displaying Multiple banners in grid layout, having Unlimited impressions,  and to offer Advertising for sale that are available in the premium version. To see all the features available in the premium versions, check out the [live demo server](http://demo.wpadcenter.com/).

The <a href="http://club.wpeka.com/product/wpadcenter/">PRO version</a> of the plugin is also available. 

<b>It features:</b>

* Geo Targeting Adzones based on countries
* Unlimited Adzones / Banner Sizes
* Unlimited Campaigns
* Easy Ad Zone Placement
* Easy to Add your Media
* Detailed Statistics
* Offer Advertising for Sale
* Automatically Collect Payment
* Advertisers Can View Their Stats Online

Experience the PRO version today.

== Installation ==

1. Upload the `wp-adcenter-lite` folder and all its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create new Adzones,Campaigns,Advertisers from the “WP Ad Center » Settings” menu in the control panel sidebar
